Project Management for Instructional Design: Managing Faculty Partnerships and Project Scope

Welcome!

Welcome to the Project Management for Instructional Design course! This course is designed to guide you through key concepts and practical exercises of applying project management elements to course development projects.

Course structure

Module Description
Module 1: Introduction Explores foundational project management concepts for instructional design.
Module 2: Timeline and Project Scope Determining project scope and developing realistic timelines to project completion.
Module 3: Managing Faculty Relationships Strategies for collaborating with faculty and maintaining positive relationships.
Final Assessment Hands-on case study to apply project management skills.
