Syllabus

Course Objectives

By the end of this course, you will be able to:

  • Identify essential project management components for instructional design projects.
  • Draft realistic project timelines by estimating completion dates for course development tasks using faculty constraint data.
  • Distinguish between in-scope and out-of-scope requests.
  • Select appropriate responses to faculty deadline conflicts.
  • Create a basic project scope statement.
  • Develop a basic project management plan for a course development scenario.
